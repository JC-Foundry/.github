# JC Foundry

Independant software development by John Craik.

JC Foundry is my professional development umbrella for client websites, reusable .NET packages, internal tools, and selected portfolio projects.

This organisation exists to keep my public work separate from personal experiments, old prototypes, and private work-in-progress repositories.

## What lives here

### Client Projects

Production client work developed under the JC Foundry name.

Some client repositories may be private depending on the project, client requirements, or deployment setup.

### JC Packages

Reusable .NET packages extracted from real projects and built to support future applications.

These packages are intended to reduce duplicated code across my own projects and, where useful, provide public examples of practical library design.

### Portfolio Projects

Larger software projects used to demonstrate real-world development, architecture, deployment, and problem-solving.

These may include full-stack applications, internal admin systems, infrastructure tooling, and long-running personal projects.

## Main Areas

* ASP.NET Core web applications
* Razor Pages and MVC-style application development
* Entity Framework Core and SQL Server
* Reusable .NET libraries and NuGet packages
* Microsoft 365 / Exchange Online integrations
* GitHub API integrations
* Admin portals and internal tools
* Cloudflare-hosted web applications
* Home-lab and infrastructure-driven software

## Featured Projects

### Ultimate Monopoly / Monappoly

A full-stack online board game project built around custom game logic, player accounts, real-time gameplay flows, admin tooling, issue reporting, and long-term maintainability.

### Alwaha London

This project was developed for a client and included both a public facing website and a management website, which is used to dynamically update the menu on the public site.

### VPN Portal

An internal admin portal for managing WireGuard VPN peers on a MikroTik-based network.

This is a portfolio case study and real operational tool, not a plug-and-play public VPN platform.

### JC Packages

A growing collection of reusable .NET packages used across my applications, including shared foundations, communication services, and GitHub integration tooling.

## About

JC Foundry is currently a one-person development label run by John Craik.

The focus is practical software: applications that solve real problems, packages that remove repeated work, and projects that demonstrate how systems are planned, built, deployed, and maintained.

Personal experiments, abandoned prototypes, and rough work-in-progress repositories usually remain outside this organisation.
